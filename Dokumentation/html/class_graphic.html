Detailed Description

Die Klasse Graphic dient als zentrales Objekt, wenn es um das Laden und Darstellen von Sprites geht.

Alle Grafiken werden in einer Map organisiert und mit Hilfe der blitSurface Methode auf dem Screen angebracht
