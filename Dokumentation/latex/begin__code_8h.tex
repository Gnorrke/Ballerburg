\subsection{Detailed Description}
This file sets things up for C dynamic library function definitions, static inlined functions, and structures aligned at 4-\/byte alignment. 

Force structure packing at 4 byte alignment.

This shouldn\textquotesingle{}t be nested -- included it around code only.

If you don\textquotesingle{}t like ugly C preprocessor code, don\textquotesingle{}t look at this file. \+:)

This is necessary if the header is included in code which has structure packing set to an alternate value, say for loading structures from disk. The packing is reset to the previous value in \doxyref{close\+\_\+code.\+h}{p.}{close__code_8h}
